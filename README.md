### Personal Website
Personal website - there's really no more description to note other than that. Technology stack is simply basic combination of HTML, CSS and Javascript.

Favicon is sourced from Favicon.cc (https://www.favicon.cc/?action=icon&file_id=824099)

Architecture:
- Normal HTML, CSS and Javascript for majority of the Website
- Sanity to handle photo gallery (acting as the CMS: content-management-system)

For local testing, use:
`python3 -m http.server 8080`
